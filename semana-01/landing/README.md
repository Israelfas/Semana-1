# Reflexión del Proyecto - Semana 01

## ¿Qué aprendimos?

En este proyecto aprendimos a usar HTML semántico para organizar mejor una página web. Entendimos que no es lo mismo usar solo div que usar etiquetas como header, main, section o footer, ya que estas le dan más sentido al contenido.

También aprendimos a usar nuevas etiquetas que no conocíamos como article, dl, details, meter y progress. Además vimos cómo insertar videos y contenido externo con iframe.

---

## ¿Qué dificultades tuvimos?

Al inicio nos confundimos con algunas etiquetas, especialmente la diferencia entre section y article.

También tuvimos problemas con Git, ya que no organizamos bien los archivos desde el principio y tuvimos que corregir la estructura del repositorio.

Otra dificultad fue que algunos recursos como imágenes y videos no cargaban, por lo que tuvimos que cambiarlos.

---

## ¿Cómo lo solucionamos?

Buscamos información en internet y revisamos ejemplos para entender mejor las etiquetas.

En Git, reorganizamos las carpetas y volvimos a hacer commits para dejar todo en orden.

Para los videos e imágenes, usamos opciones más simples y seguras para que no fallen.

---

## ¿Qué mejoraríamos?

Nos gustaría mejorar el diseño con CSS, ya que la página solo tiene estructura.

También podríamos usar recursos multimedia reales en lugar de ejemplos.

Además, podríamos mejorar un poco más la accesibilidad.

---

## Trabajo en equipo

El trabajo se hizo en grupo, cada uno ayudó en diferentes partes como el código, la organización y los commits.

Nos comunicamos para resolver los errores y poder avanzar.

---

## Conclusión

Este proyecto nos ayudó a entender mejor cómo estructurar una página web con HTML semántico.

También aprendimos a usar Git para trabajar en equipo y a corregir errores durante el desarrollo.
